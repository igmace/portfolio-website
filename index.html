My Work

MODERNfinds | Online Store Web App

The goal of this project was to design a responsive web app for an online store that allowed users to view inventory on-the-go and filter the search to their needs. I built intuitive user flows to guide them through the app and help them reach their goal of making a purchase. I also established the brand guidelines to create a consistent visual and content style.

Skills: Branding, Responsive Design, User Stories, User Flows, User Testing, Wireframing, Prototyping, UI Patterns, UX Research, Mockups

Casa | Home Maintenance Native App

Casa is a native app for iOS and Android that follows the design guidelines for each system. I began by conducting interviews to learn the pain points users had with existing apps. I moved on to create wireframes and prototypes for testing. After implementing the feedback from my users, I created the style guide, the final wireframes and mockups.

Skills: Native Design, UX Research, User Personas, User Flows, User Stories, User Testing, Wireframing, Prototyping, Style Guide, UI Patterns, Mockups
